# ProjetoEnquetes
Sistema de enquetes

Funcionalidades:
CRUD(CREATE, READ, UPDATE DELETE)
Responder formulário de perguntas e respostas.
Gerar gráficos através da API HighChart Graph
Fazer download dos gráficos gerados.

Tecnologias utilizadas:

Java
Thymeleaf
JPA
DevTools
Web
SpringBoot
Hibernate
HTML
CSS
Express js
PostgreSQL
